Without clear

Element 1
Element 2 - Notice that Element 2 is after Element 1 in the HTML code. However, since Element 1 floats to the left, the text in Element 2 flows around Element 1.


With clear

Element 1
Element 2 - Here, clear: left; moves Element 2 down and below the floating Element 1. The value "left" clears elements floated to the left. You can also clear "right" and "both".
